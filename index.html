ЦВЕТА (переменные)

:root {
    --c: #FFF2DF;   /* кремовый — фон */
    --o: #F6D5A5;   /* оранжевый — секции, футер */
    --m: #CCEDEE;   /* мятный — карточки, кнопки */
    --g: #333;      /* серый — текст */
    --w: #fff;      /* белый */
    --r: 12px;      /* радиус скругления */
}

БАЗА (сброс)

* { margin: 0; padding: 0; box-sizing: border-box; }

🔧 Git — Команды

🟢 Первый раз (создание репозитория)

git init
git add .
git commit -m "Все файлы проекта"
git remote add origin https://github.com/логин/репо.git
git push -u origin master

🟡 После изменений

git add .
git commit -m "Изменил CSS"
git push

🔵 Полезные команды

Команда Что делает
git status Что изменено
git log История коммитов
git pull Скачать изменения
git clone URL Скачать репозиторий

📦 Коммиты по модулям (для экзамена)
